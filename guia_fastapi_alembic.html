🔌 Banco de Dados

db/session.py:

from sqlalchemy import create_engine
from sqlalchemy.orm import sessionmaker

DATABASE_URL = "sqlite:///./test.db"
engine = create_engine(DATABASE_URL, connect_args={"check_same_thread": False})
SessionLocal = sessionmaker(bind=engine, autoflush=False, autocommit=False)

db/base.py:

from sqlalchemy.orm import declarative_base
Base = declarative_base()

🧱 Modelo

from sqlalchemy import Column, Integer, String
from app.db.base import Base

class User(Base):
    __tablename__ = 'users'
    id = Column(Integer, primary_key=True, index=True)
    name = Column(String)
    email = Column(String, unique=True)

🚀 Rotas e Inicialização

# main.py
from fastapi import FastAPI
from app.routers import user
app = FastAPI()
app.include_router(user.router)
# routers/user.py
from fastapi import APIRouter, Depends
from sqlalchemy.orm import Session
from app.db.session import SessionLocal
from app.models.user import User

router = APIRouter(prefix="/users", tags=["Users"])

def get_db():
    db = SessionLocal()
    try: yield db
    finally: db.close()

@router.get("/")
def get_users(db: Session = Depends(get_db)):
    return db.query(User).all()

🧬 Alembic

Inicialização:

alembic init alembic

Editar alembic.ini:

sqlalchemy.url = sqlite:///./test.db

Editar env.py:

from app.db.base import Base
from app.models import user

target_metadata = Base.metadata

Criar migration:

alembic revision --autogenerate -m "create user table"

Aplicar:

alembic upgrade head

📚 Comandos Alembic

  • alembic history - ver histórico
  • alembic current - versão atual
  • alembic upgrade head - aplicar tudo
  • alembic downgrade -1 - reverter última

🛡️ Segurança

  • Use pydantic para validar entradas
  • Use python-dotenv para variáveis sensíveis
  • Habilite CORS:
from fastapi.middleware.cors import CORSMiddleware
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_methods=["*"],
    allow_headers=["*"],
)

📦 Extras

  • Separação por módulos: crud, schemas, routers
  • Versionamento com prefixo nas rotas: /api/v1
